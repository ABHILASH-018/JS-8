# Service Booking App

## Project Description

This is a simple Service Booking App created using HTML, CSS, and JavaScript.

The application allows users to:

- View available services
- Add services to the cart
- Remove services from the cart
- Calculate the total amount
- Fill out a booking form
- Save the cart using Local Storage
- Complete a booking

This project was built to practice JavaScript concepts like arrays, functions, loops, DOM manipulation, events, and Local Storage.

## How to Run

1. Download or clone the project.
2. Make sure all four files are in the same folder.
3. Open **index.html** in any web browser.
4. Browse the services.
5. Click **Add** to add services.
6. Check the cart and total amount.
7. Fill in the booking form.
8. Click **Book Service** to complete the booking.
